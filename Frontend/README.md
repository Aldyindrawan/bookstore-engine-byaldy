# 🎨 bookStore | Frontend Engine

Bagian ini merupakan antarmuka pengguna (UI/UX) dari aplikasi **bookStore** yang dibangun menggunakan keandalan **React.js** dengan bantuan build tool **Vite** untuk performa pengembangan yang cepat dan efisien.

### ⚡ Tech Stack & Tools:
* **React.js (Vite):** Library utama untuk membangun komponen UI yang modular.
* **Tailwind CSS:** Framework CSS untuk desain yang modern, bersih, dan responsif.
* **React Router DOM:** Menangani navigasi halaman secara dinamis (Path: `/amunisi`).
* **Axios:** Untuk menangani HTTP Request (Data Fetching) ke API Backend.
* **React Hot Toast:** Memberikan feedback notifikasi login/logout yang interaktif.
