📚 bookstore-engine-byaldy 🚀
bookStore | byAldy — Sebuah platform E-Bookstore modern yang dibangun menggunakan MERN Stack (MongoDB, Express.js, React, dan Node.js). Project ini dirancang untuk manajemen data buku secara fullstack dengan performa tinggi dan desain yang responsif.

🛠️ Tech Stack & Tools
Kombinasi teknologi yang digunakan dalam proyek ini:

Frontend: React.js (Vite) & Tailwind CSS

Backend: Node.js & Express.js

Database: MongoDB Atlas (Cloud Database)

Routing: React Router DOM (Custom Path: /amunisi)

Feedback: Dynamic Toast Notifications (Supportive User Experience)

✨ Fitur Utama
Full CRUD Operations: Manajemen inventaris buku (Tambah, Lihat, Edit, Hapus).

Supportive UX: Notifikasi login/logout dengan pesan motivasi khusus.

Scalable Architecture: Struktur folder Backend & Frontend yang terorganisir.

Modern UI: Menggunakan Tailwind CSS untuk tampilan yang bersih dan adaptif.

🚀 Cara Menjalankan Project (Local)
1. Setup Backend
Bash
cd Backend
npm install
npm run dev
2. Setup Frontend
Bash
cd Frontend
npm install
npm run dev
